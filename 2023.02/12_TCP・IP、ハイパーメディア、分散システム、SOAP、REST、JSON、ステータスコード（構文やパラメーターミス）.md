# TCP/IP
世界的に利用されている通信プロトコル。
TCP：送られてきたデータを順番に並べて、ファイルを作る
IP：データを目的地に送るもの

# 分散システム
サービスの規模が大きくなってくると、サーバーへの負荷も大きくなる。そこで、サーバーを複数使用することで負荷を分散する方法のこと。

# SOAP
HTTPをアプリケーションプロトコルではなく、トランスポートプロトコルとして扱い、HTTP上で独自のメッセージを転送するプロトコル。
